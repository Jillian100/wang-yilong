## 🎨 什麼是 City Pop 視覺美學？

### 定義
**City Pop** 是1980年代日本都市流行音樂的視覺風格，特點是：
- 🌆 都市夜景與霓虹燈
- 🌅 日出日落的漸變色調
- 💫 懷舊與未來感的融合
- 🎭 剪影與柔焦效果

### 核心美學元素

#### 1. **經典配色方案**
```css
/* Teal & Orange（青橙對比）*/
--teal: #4ECDC4;           /* 青綠色 - 陰影與背景 */
--orange: #FF9B6D;         /* 溫暖橘 - 光源與高光 */

/* 輔助色 */
--soft-pink: #FFB4C8;      /* 柔粉紅 - 浪漫氛圍 */
--lavender: #9B8CD8;       /* 薰衣草紫 - 夢幻感 */
--golden-yellow: #FFD54F;  /* 金黃色 - 溫暖光芒 */
--deep-blue: #2D3E82;      /* 深邃藍 - 夜空與深度 */
```

**配色理論**：
- Teal & Orange 是電影級配色（互補色對比）
- 冷暖對比創造視覺張力
- 柔和的飽和度營造懷舊感

#### 2. **視覺風格關鍵字**
```
時代特徵：
- Retro 1980s Japanese aesthetic
- Nostalgic / Vintage feeling
- Vaporwave fusion

技術特徵：
- Soft focus
- Dreamy bokeh
- Volumetric lighting
- Lens flares
- Film grain (optional)

畫風：
- Retro anime painting style
- Shoujo manga aesthetic
- Illustrated / Hand-painted look
```

#### 3. **典型場景元素**
```
都市場景：
✅ Urban rooftop（都市屋頂）
✅ City skyline（城市天際線）
✅ Neon signs（霓虹招牌）
✅ Street lights（街燈）
✅ Retro cafe（復古咖啡廳）

自然場景：
✅ Seaside pier（海邊碼頭）
✅ Sunset / Sunrise（日落日出）
✅ Twilight / Golden hour（黃昏金色時刻）

人物元素：
✅ Silhouettes（剪影）- 強調氛圍而非細節
✅ Group of friends（朋友群像）
✅ Solitary figure（孤獨身影）
```

## 🔧 Midjourney 提示詞技巧

### 成功公式

基於 10組提示詞的實戰經驗，提煉出以下公式：

```
[場景描述] + [視覺元素] + [配色方案] + [情感氛圍] + [藝術風格] + [技術參數]
```

### 分解示例

**奏效的提示詞**（Urban rooftop 系列）：
```
City pop night scene digital art,
urban rooftop at twilight with warm string lights,
group of friends silhouettes sharing moment together,
nostalgic 80s color grading with teal and orange tones,
soft focus background with city lights bokeh,
intimate and grateful atmosphere,
retro anime painting style,
--ar 16:9 --style raw --stylize 350
```

**分解分析**：
```
1. 場景描述：
   "City pop night scene digital art"
   → 明確風格定位

2. 具體場景：
   "urban rooftop at twilight with warm string lights"
   → 具體而非抽象

3. 人物元素：
   "group of friends silhouettes sharing moment together"
   → 剪影 + 情感敘事

4. 配色方案：
   "nostalgic 80s color grading with teal and orange tones"
   → 明確指定配色

5. 技術效果：
   "soft focus background with city lights bokeh"
   → 柔焦 + 光斑

6. 情感氛圍：
   "intimate and grateful atmosphere"
   → 情感基調

7. 藝術風格：
   "retro anime painting style"
   → 畫風定位

8. 技術參數：
   "--ar 16:9 --style raw --stylize 350"
   → 比例、風格、藝術化程度
```

## 💡 從歌詞到視覺的轉化方法

### 轉化流程

```
歌詞意象 → 核心情感 → 視覺隱喻 → 具體場景 → Midjourney 提示詞
```

### 實戰案例：《謝謝有你們》

#### Step 1: 提取歌詞意象
```
關鍵歌詞意象：
1. "在天最黑暗的時候 點亮一盞燈"
2. "像風 吹拂著白云 總是不著痕跡"
3. "把幸福悄悄放進我的手心"
4. "我合手祈禱天明 願陽光可以照進每個人的心"
5. "聽風 一絲絲笑意叮叮噹噹聲音"
```

#### Step 2: 識別核心情感
```
核心情感：
- 感恩（Grateful）
- 陪伴（Companionship）
- 溫暖（Warmth）
- 希望（Hope）
```

#### Step 3: 視覺隱喻轉化表

| 歌詞意象 | 核心情感 | 視覺隱喻 | 具體場景元素 |
|---------|---------|---------|-------------|
| 黑暗中的燈 | 希望、溫暖 | 霓虹燈、街燈、燭光 | rainy urban street, glowing storefront windows |
| 風吹白雲 | 輕盈、溫柔 | 微風、飄動、海邊 | gentle breeze, flowing clothes, seaside pier |
| 手心的幸福 | 親密、珍惜 | 捧光、雙手、光球 | hands holding glowing orb, warm light |
| 祈禱天明 | 希望、信念 | 破曉、陽台、合手 | dawn, balcony, prayer, sunrise |
| 笑意鈴聲 | 愉悅、輕快 | 風鈴、音符、輕盈 | wind chimes, musical notes, floating |

#### Step 4: 構建提示詞

**範例：「黑暗中的燈」→ 雨夜街道場景**
```
City pop digital painting,
lone figure walking through rainy urban street at night,
warm glowing storefront windows reflecting on wet pavement,
umbrella silhouette against neon signs,
nostalgic 80s color palette with deep blues and amber golds,
soft rain bokeh effects,
hopeful and contemplative mood,
retro Japanese illustration style,
cinematic lighting,
--ar 16:9 --style raw --stylize 350
```

**關鍵轉化技巧**：
1. ✅ **保留情感本質** - "黑暗中的燈"變成"warm glowing storefront windows"
2. ✅ **增加敘事性** - 加入"lone figure walking"創造故事感
3. ✅ **視覺細節化** - "reflecting on wet pavement"增加真實感
4. ✅ **情感氛圍具體化** - "hopeful and contemplative mood"

## ✅ 提示詞優化技巧

### 有效做法 ✅

#### 1. **具體場景描述**
```
❌ 錯誤："beautiful city at night"
✅ 正確："urban rooftop at twilight with warm string lights"
```
**理由**：具體勝過抽象，Midjourney 更容易理解

#### 2. **明確情感氛圍**
```
❌ 錯誤："nice feeling"
✅ 正確："intimate and grateful atmosphere"
```
**理由**：情感詞彙影響色調和光線

#### 3. **技術風格標籤**
```
❌ 錯誤："looks like old Japanese art"
✅ 正確："retro anime painting style, nostalgic 80s aesthetic"
```
**理由**：使用 Midjourney 熟悉的風格術語

#### 4. **光線效果強調**
```
✅ 添加光線描述：
- "soft focus background with city lights bokeh"
- "volumetric lighting"
- "lens flares"
- "warm golden light"
```
**理由**：光線決定氛圍，City Pop 尤其依賴光效

#### 5. **剪影優於細節**
```
✅ 使用剪影：
- "group of friends silhouettes"
- "solitary figure silhouette"
```
**理由**：剪影留白讓觀者投射情感，更有共鳴

---

### 避免事項 ❌

#### 1. **過於抽象**
```
❌ 避免："beautiful", "amazing", "perfect"
```
**原因**：無法給 AI 具體指引

#### 2. **缺乏時代定位**
```
❌ 避免：只說"city"而不說"80s"
✅ 改進："nostalgic 80s color grading"
```

#### 3. **忽略情感基調**
```
❌ 避免：只有視覺沒有情感
✅ 改進：加入"intimate", "grateful", "hopeful"
```

#### 4. **過度複雜**
```
❌ 避免：一次塞太多元素
✅ 改進：一個場景一個焦點
```

#### 5. **使用 artist: 前綴（對某些查詢）**
```
❌ 經驗證對 Spotify 搜尋：artist:王一隆 會失敗
✅ 直接使用名字：王一隆
```
**註**：這是 Spotify API 的經驗，可能不適用 Midjourney

## 🎯 Midjourney 參數指南

### 常用參數

#### 比例 (Aspect Ratio)
```
--ar 1:1     → 正方形（專輯封面、IG Post）
--ar 4:5     → 縱向（單曲封面、手機壁紙）
--ar 16:9    → 橫向（MV 場景、YouTube）
--ar 21:9    → 超寬（電影感海報）
--ar 9:16    → 直式（IG Story、手機全屏）
```

#### 風格控制
```
--style raw        → 更精準控制，減少 MJ 預設美化
--style expressive → 更藝術化，適合創意探索
```

#### 藝術化程度 (Stylize)
```
--stylize 50-100   → 寫實，適合需要具體形象
--stylize 250-350  → 平衡（City Pop 推薦）⭐
--stylize 500-750  → 高度藝術化
--stylize 1000     → 極度風格化
```

**City Pop 推薦設定**：
```
--ar 16:9 --style raw --stylize 300-350
```

## 🔄 迭代優化流程

### 提示詞優化循環

```
1. 初版提示詞
   ↓
2. 生成圖像（4張）
   ↓
3. 評估結果
   - 配色正確嗎？
   - 氛圍對了嗎？
   - 構圖平衡嗎？
   ↓
4. 識別問題
   - 太亮/太暗？ → 調整光線描述
   - 不夠懷舊？ → 加強"nostalgic", "80s"
   - 過於寫實？ → 增加"painting style"
   ↓
5. 修正提示詞
   ↓
6. 重新生成
   ↓
7. 比較前後差異
   ↓
8. 記錄有效的調整
```

## 💎 最佳實踐總結

### 黃金法則

#### 1. **情感先行**
```
不是："我要一張城市夜景"
而是："我要傳達溫暖陪伴的感恩之情，背景是城市夜景"
```

#### 2. **具體勝過抽象**
```
不是："beautiful scene"
而是："urban rooftop at twilight with warm string lights"
```

#### 3. **配色必須明確**
```
不是："good colors"
而是："teal and orange tones, nostalgic 80s color grading"
```

#### 4. **一個焦點**
```
不是：塞滿各種元素
而是：一個場景一個核心意象
```

#### 5. **保留測試記錄**
```
不是：只保留成功的
而是：記錄失敗和成功，分析差異
```
