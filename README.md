## 🎨 美學基因庫

F_web_design 維護一套完整的**美學 DNA**，確保所有項目的視覺一致性：

### 色彩系統
- 主色調：極簡黑白
- 點綴色：Purple (#667eea), Blue (#4299e1), Cyan (#0bc5ea)
- 功能色：Success, Warning, Error, Info

### 字體系統
- 中文：Noto Sans TC / Noto Serif TC
- 英文：Inter, Playfair Display
- 代碼：JetBrains Mono

### 間距系統
- 基於 8px 系統
- 從 4px 到 96px 的完整規範

### 動畫系統
- 緩動函數：cubic-bezier
- 標準動畫：fadeInUp, scaleIn
- 持續時間：150ms / 300ms / 500ms / 800ms

詳見：[docs/AESTHETIC_DNA.md](docs/AESTHETIC_DNA.md)

## 🎯 品質標準

### 美學評分 (0-10)
- 10分 - 完美無瑕，如極光般驚艷
- 9分 - 優秀，小瑕疵可接受
- 8分 - 良好，符合標準
- 7分 - 合格，需要優化

### 評分維度
- 視覺美感 (30%)
- 用戶體驗 (25%)
- 技術實現 (25%)
- 創新性 (10%)
- 品牌一致性 (10%)

### 性能標準
- Lighthouse Score: 90+
- First Contentful Paint: < 1.5s
- Largest Contentful Paint: < 2.5s
- Cumulative Layout Shift: < 0.1
